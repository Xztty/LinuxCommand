
# md5sum/sha1sum

## md5sum介绍

计算/check md5信息摘要, 结果为128bit，32个16进制字符，如：b1946ac92492d2347c6235b4d2611184

### 示例

```text
md5sum a.log
echo -n "hello" | md5sum     # 计算字符串的md5
```

## sha1sum介绍

计算/check md5信息摘要, 结果为140bit，40个16进制字符，如：b30c2130e7a42f7762d5e7582d94c4ea45d1de9c

## 示例

```text
sha1sum a.log
echo -n "hello" | sha1sum
```
