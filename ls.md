
# ls

## 介绍

查看目录的内容，文件列表信息

## 常用示例

```bash
ls                     # 默认按照字典序排序
ls -al                 # 显示所有隐藏文件
ls -il                 # 显示文件和inode编号
ls -tl                 # 显示文件以最后修改时间排序
ls -L                  # 显示软链接所指向的文件信息
ls -S                  # 以文件大小排序
ls -r                  # reverse order
ls -ld -- */           # show only directory
```
