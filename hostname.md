
# hostname 

## 介绍

查看修改hostname

## 示例

```text
hostname                         # 查看hostname
hostname -I                      # 查看hostname的所有ip
hostname -F /etc/hostname        # 从文件读取hostname
hostname test                    # 临时设置hostname
```
