
# shutdown/reboot 

## shutdown介绍

关闭系统，用于多用户场景，会发送消息给所有登录的用户

### 示例

```text
shutdown                 # 关闭系统及电源
shutdown -r              # 重启
shutdown now
shutdown +60             # 60min后关闭
```
