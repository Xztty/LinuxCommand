
# tree 

## 介绍

以树形展示文件夹内容

## 示例

```text
tree                   #
tree -d                # 只显示文件夹
tree -f                # 路径显示完整
tree -P "*.txt"        # 通配符过滤，只显示满足匹配项
tree -C                # 文件和文件夹不同颜色
tree -l                # 遇到软链接继续进去显示
tree -L 3              # 显示层数，层数从1开始
```
